# Revision Process

## Pass 1: Big Picture
- [ ] Does the story make sense?
- [ ] Do character arcs land?
- [ ] Are all promises fulfilled?

## Pass 2: Line Editing
- [ ] Cut filler words
- [ ] Tighten prose
- [ ] Improve clarity

## Pass 3: Proofreading
- [ ] Spelling/Grammar
- [ ] Consistency
- [ ] Style/Voice

## Beta Reader Checklist
- [ ] Characters relatable?
- [ ] Any confusing scenes?
- [ ] What did you enjoy most?
